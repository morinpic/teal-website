## 1. ブランドポジション

### 現在地と目標

横浜元町エリアの競合分析から、teal. は以下のポジションを目指す。

```
               ハイエンド
                   ↑
    KINGDOM ─────┼──── ★ teal.（目指す場所）
    実績・技術訴求  │        ブランドストーリー
                 │        × デジタル体験品質
AgoraLuludi ────┼
空間・体験訴求    │
                   ↓
               カジュアル
```

- **KINGDOM**（実績・権威訴求）と **AgoraLuludi**（空間・体験訴求）の間に、**ブランドストーリー × デジタル体験の質**で差別化できるポジションがある
- 「なぜteal.か」という**理由を持たせる**ブランド訴求が主流化する中で、技術力・スタッフのストーリーを前面に出す方向性が競争優位になる

## 2. 2026年デザイントレンド 戦略的採用方針

### 採用するトレンド（teal.との親和性が高いもの）

#### ① セリフ体ヘッダーの導入
- **背景**: AI生成コンテンツの氾濫で「無機質なゴシック体」に疲弊感。serif / high-contrast フォントが高級・誠実さのシグナルとして再評価されている
- **teal.への適用**: 見出しレベル（H1〜H2）にセリフ体を導入し、ボディはゴシック体を維持するコントラスト構成
- **推奨フォント候補**:
  - 日本語: `Noto Serif JP`（Google Fonts, 無料）
  - 英語アクセント: `Cormorant Garamond` / `Playfair Display`
- **優先度**: ★★★★★（即効性が最も高い）

#### ② ナチュラル・オーガニックテクスチャ
- **背景**: グレイン（粒状感）、クリーム・オフホワイト背景、自然素材感が高級感と相性◎
- **teal.への適用**: 純白(`#ffffff`)から**クリームホワイト**(`#fafaf8` 程度)への移行。セクション区切りに薄いテクスチャを活用
- **ブランドカラー `#0c7c7c` との相性**: 緑系ティールはオーガニック・ナチュラルの方向性と完全一致
- **優先度**: ★★★★☆

#### ③ スクロール演出のクオリティアップ
- **背景**: 単純フェードインはもう標準装備。タイミング・イージングの質でブランドの格が出る時代
- **teal.への適用**: cubic-bezier を luxury トーンに調整。ステガード（段階的）アニメーションで奥行き感を演出
- **優先度**: ★★★★☆

#### ④ エフォートレス（Effortless）な写真世界観
- **背景**: 「作りすぎない自然な仕上がり」がヘアサロン業界の最新トレンドキーワード
- **teal.への適用**: サイトに掲載するビジュアルも「自然体の上質感」を基準に統一。加工過多な写真より素材感・光・空間を活かしたものを優先
- **優先度**: ★★★☆☆（コンテンツ制作側との連携が必要）

### 採用しないトレンド（理由付き）

| トレンド | 見送り理由 |
|---------|----------|
| ブルータリズム | teal.の上品・自然路線と相反する |
| ダークモード | Phase 2では保留（D-14）。世界観構築を優先 |
| ネオン・大胆カラー | ブランドカラーの静謐な上質感を損なう |
| グリッドの可視化 | カジュアル・テック印象になりすぎる |

## 3. ビジュアルデザイン方針

### カラーシステム方針（Phase 2）

```
Primary:     #0c7c7c  （ティールグリーン ← 変更なし）
Background:  #fafaf8  （クリームホワイト ← #ffffff から移行検討）
Dark Text:   #231815  （ダークブラウン ← 変更なし）
Accent:      #0c7c7c 10%透過（セクション背景に使用）
```

> ※ #fafaf8 への移行は全体への影響が大きいため、デザインレビュー後に正式決定する

### タイポグラフィ方針（Phase 2）

```
見出し H1-H2:  Noto Serif JP 400/500 + 游明朝 fallback
見出し H3-H4:  現行の游ゴシック / Noto Sans JP 維持
本文:           現行のゴシック体維持（可読性優先）
英語アクセント: Cormorant Garamond（ブランド名・キャッチコピー）
```

### レスポンシブ方針（Phase 2）

- **モバイルファースト再設計**（現状：PC優先で作られている可能性）
- **CTAボタン**: タップ領域最低 44px × 44px を徹底
- **フォントサイズ**: モバイルで H1 36px 以上、本文 16px 以上

## 4. コンテンツ戦略方針

### 「理由で選ぶ」ブランド訴求

2026年の美容業界ではブランドを「理由」で選ぶ行動が主流になる。teal.サイトに必要な要素：

1. **技術の根拠を見せる** — スタッフ紹介にスキル・得意施術・こだわりを追加（C-03）
2. **お客様の声・ビフォーアフター** — エフォートレスな結果を見せるコンテンツ
3. **サロンの哲学・ストーリー** — なぜ横浜元町にteal.なのか

### Instagram 連携方針（C-04）

- サイトの世界観とInstagramのビジュアルを統一する
- 「エフォートレス」な投稿スタイルへの移行をサイト側から先導
