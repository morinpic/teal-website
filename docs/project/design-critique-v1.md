## 各軸の詳細評価

### 1. ユーザビリティ（Nielsen原則）— 7.5/10

**良い点:**
- システム状態の可視性: スクロールで Header にシャドウが付き、現在位置がわかる
- ユーザーコントロール: ScrollToTop ボタンがあり脱出導線あり
- 一貫性: ボタン・リンクのスタイルが統一されている（Button.tsx コンポーネント）
- エラー防止: ESLintで型チェック徹底、CMS連携でコンテンツミス防止

**課題点:**
- **ヘルプと文書化が不足**: FAQ セクションは存在するが、予約の流れ・キャンセルポリシーなどが見つかりにくい
- **認識より想起**: ナビゲーションが 8 項目と多く、初回訪問者が重要な情報にたどり着くまでのステップが多い
- **エラーからの回復**: 404ページは実装されているが、デザインの質が不明

---

### 2. 視覚的ヒエラルキー — 6.5/10

**良い点:**
- SectionHeading コンポーネントで各セクションのタイトル・サブタイトル構成が統一
- AnimatedLine でセクションタイトルに視覚的なアクセントあり
- ヒーローの 60% ダークオーバーレイで白テキストの視認性確保

**課題点（重大）:**
- **全セクションが同じ重さで並ぶ**: STYLE, MENU, STAFF, NEWS, BLOG など、情報の優先度がビジュアル上で差別化されていない。ユーザーが「何を見てほしいサイトか」が伝わりにくい
- **ヒーロー後の引きが弱い**: ヒーローの次のセクションへの流れが唐突。ブランドストーリーや About 導線が弱い
- **見出しサイズの振れ幅が小さい**: `text-3xl lg:text-4xl` 固定で、インパクトのある大型タイポグラフィがない

---

### 3. タイポグラフィ — 6.0/10

**良い点:**
- Noto Sans JP を一本化しており、日本語の可読性は担保されている
- `tracking-widest` のラベルテキストはブランドの洗練感に貢献
- 行間 1.9 は本文の読みやすさとして適切

**課題点（重大）:**
- **フォントが1種類のみ**: Noto Sans JP のみではブランドの「格」が出ない。2026年トレンドではセリフ体の復権が顕著で、競合との差別化機会を逃している
- **タイポグラフィのコントラストが不足**: 見出し・サブタイトル・本文の重さの差が小さく、スキャナビリティ（流し読みしやすさ）が低い
- **英語テキストの扱いが中途半端**: 「hair salon / yokohama motomachi」など英語テキストがあるが、専用の英語フォントが当たっておらず質感が落ちている

---

### 4. カラーシステム — 8.0/10

**良い点:**
- `#0c7c7c` ティールグリーンは横浜・元町の雰囲気と相性が良く、記憶に残るカラーアイデンティティ
- teal-primary / teal-dark / teal-light の3段階バリエーションが定義されている
- ダークテキスト `#1a1a1a` は読みやすく、暖かみのある黒として機能している
- WCAG AA準拠（コントラスト比 5.0:1 確認済み）

**課題点（軽微）:**
- **純白背景 `#ffffff` がブランドとやや乖離**: 2026年トレンドのオーガニック・ナチュラル路線（クリームホワイト `#fafaf8`）への移行で、より上質感が増す可能性がある
- **ゴールドアクセント `#d4af37` が未活用**: 定義されているが実際のUIに使われていない。削除か活用の方針決定が必要
- **グレーの階調が不足**: `text-dark-text/50` のような透過処理に頼りすぎており、意図的なグレースケールパレットが薄い

---

### 5. モバイルUX — 7.0/10

**良い点:**
- FloatingReserve ボタン（モバイル専用スティッキーバー）は予約導線として機能的
- Safe area inset padding でノッチ対応済み
- フルスクリーンオーバーレイメニューは操作しやすい

**課題点:**
- **タップターゲットサイズの確認が必要**: ナビゲーションリンクが `text-xs` サイズで、44px × 44px の最小タップ領域を満たしているか疑問
- **モバイルでの本文フォントサイズ**: `text-sm` （14px）箇所が残っていると読みにくい。最小 16px が推奨
- **スクロール演出のモバイル最適化**: Framer Motion のアニメーションがモバイルで重くなる可能性がある（特に低スペック端末）

---

### 6. ブランド戦略一貫性 — 6.5/10

**良い点:**
- ブランドカラーの一貫適用
- Button.tsx による CTA スタイルの統一
- フォントの一本化

**課題点（重大）:**
- **「teal.らしさ」のビジュアル言語が確立していない**: 競合の KINGDOM（権威）・AgoraLuludi（空間体験）と比べ、teal. 独自の世界観がサイトから伝わりにくい。コンセプトが「普通の美容室サイト」の域を出ていない
- **ブランドストーリーが不在**: 「なぜ横浜元町にteal.なのか」「どんな想いで営業しているか」が伝わるコンテンツが現状ない
- **スタッフの個性が見えない**: スタッフ一覧はあるが、技術や人柄が伝わる情報が薄い

---

### 7. 認知負荷 — 7.5/10

**良い点:**
- セクションごとにコンテンツが整理されており、情報過多ではない
- MenuSection のタブUIで料金表が整理されている
- アニメーションが `once: true` なので繰り返し動いてうるさくない

**課題点:**
- **ナビゲーション項目が多い（8項目）**: ABOUT / STYLE / MENU / STAFF / FAQ / NEWS / BLOG / ACCESS と多く、初回訪問者への認知負荷が高い。3〜5項目程度に絞るか、グルーピングが望ましい
- **Hero の CTA が2つ**: 「予約する」と「Scroll Down」が並んでいる。後者は UX 的に不要な可能性

---

### 8. アクセシビリティ（WCAG AA） — 8.0/10

**良い点:**
- スキップリンク実装済み
- focus-visible スタイル実装済み（keyboard navigation 対応）
- HairSalon 構造化データ（schema.org）実装
- aria-label がハンバーガーボタン等に設定されている

**課題点（軽微）:**
- **インタラクティブ要素の alt テキスト**: 装飾的画像と情報的画像の区別・alt 記述の質確認が必要
- **カラーのみに依存した情報伝達がないか確認**: teal 色のボタンが無効状態になった場合のアクセシビリティ
- **モーション**: `prefers-reduced-motion` への対応確認が必要（Framer Motion の設定）

---

### 9. インタラクション・モーション — 7.5/10

**良い点:**
- カスタムイージング `[0.16, 1, 0.3, 1]` が premium 感に貢献している
- スタガー（段差）アニメーションがリスト表示に使われており、視線誘導に機能している
- ヒーローの段階的出現（0.3s/0.6s/0.9s/1.2s）は洗練されている

**課題点:**
- **モーションの「語彙」が単一**: フェードイン + Y-オフセットのみ。2026年のハイエンドブランドは、スクロール速度に連動したパララックス、ホバー時の磁力感など、もっとリッチなモーション語彙を持っている
- **スクロールアニメーションの offset が固定（-60px）**: セクションのサイズや文脈に合わせた調整の余地がある

---

### 10. 市場差別化 — 4.0/10

**課題点（最重要）:**
- **2026年の美容室サイトとして「普通」すぎる**: ナビ・ヒーロー・セクション積み上げという構成は業界標準テンプレートと区別がつかない
- **「エフォートレス美容」トレンドへの対応ゼロ**: 現在のサイトデザインからは、teal. が提供するスタイルの「雰囲気・感触」が伝わってこない
- **インスタグラムセクションが形式的**: SnsSection がプレースホルダー表示のままであれば、むしろブランドイメージを下げるリスクがある
- **競合との差異がサイトから読み取れない**: なぜ KINGDOM や他の元町美容室ではなく teal. なのかが、サイトを見ても答えられない

---

## 改善分類

### 🔴 Critical（ブランド知覚を阻害している）

1. **タイポグラフィの単一フォント問題** → セリフ体の見出し導入で格が変わる（No.10）
2. **ブランドストーリーの不在** → サイトを見た人が「teal.らしさ」を言語化できない（No.16 + コンテンツ戦略）
3. **市場差別化の欠如** → 業界テンプレートから脱却するビジュアル言語の確立が急務

### 🟡 Important（改善効果が大きい）

4. **視覚的ヒエラルキーの平坦化** → どのセクションも同じ重さで、優先度が伝わらない（視覚設計の見直し）
5. **モバイルタップターゲット** → 予約率に直結する課題（No.9）
6. **インスタグラムセクションの機能化** → プレースホルダーのまま放置は逆効果（No.17）
7. **ナビゲーション項目の整理** → 8項目 → 5〜6項目への絞り込み

### 🟢 Polish（品質の磨き込み）

8. **背景色のクリームホワイト化** → #ffffff → #fafaf8 でオーガニック感が増す
9. **英語テキスト専用フォント** → Cormorant Garamond 等でアクセント強化
10. **ゴールドアクセントの整理** → 使わないなら CSS 変数から削除してシステムをクリーンに
11. **`prefers-reduced-motion` 対応** → アクセシビリティの完成度向上

## Phase 2 に向けた2つのデザイン代替案

### 案A: 「エディトリアル × ナチュラル」

```
コンセプト: 質の高いファッション誌のような構成
- セリフ体見出し（Noto Serif JP）× ゴシック体本文
- オフホワイト背景 #fafaf8
- 大きなビジュアル面積、テキスト最小化
- スタッフの素顔・哲学をコンテンツの核に
- スクロールで「ページをめくる」感覚
```

**メリット**: ブランドの格が一気に上がる。エフォートレス美容トレンドとも合致
**デメリット**: コンテンツ（写真・コピー）の質への依存度が高い

### 案B: 「静謐な空間体験」

```
コンセプト: 高級旅館やスパのような「間（ま）」の演出
- 極限まで余白を広げる（現状の 1.5〜2倍の padding）
- アニメーション速度を遅く（0.8〜1.2s）に変更
- テキスト量を絞り、一言に力を持たせる
- ゴシック体を保ちつつ、ウェイト・サイズのコントラストで差別化
- スクロール演出にパララックス効果を追加
```

**メリット**: コード変更のみで実現可能。コンテンツ制作が最小
**デメリット**: 案Aほどのインパクトはない。情報量が少なくなりすぎるリスク
